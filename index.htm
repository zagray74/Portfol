Über mich<

Willkommen auf der Portfolio-Seite. Mein Name ist Victor. Ich lebe in Deutschland. Ich beschäftige mich mit Layout (Website-Erstellung). Ich habe ein vollständig browserübergreifendes, adaptives Layout. Um sicherzustellen, dass die Anforderungen an das Site-Layout während der Arbeit am Projekt erfüllt werden, verwende ich den Gulp-Projektassembler zur Automatisierung des Layoutprozesses, Less/Sсss-Präprozessoren für schnelleres Site-Styling und flexibleren Code, BEM, Bootstrap und Medienabfragen für adaptive Sites Layout. Die Dynamik der Website wird durch unseren eigenen JS- oder jQuery-Code organisiert. Am Ende der Arbeit wird die Darstellung des Layouts in verschiedenen Browsern auf verschiedenen Geräten überprüft, der Code durch einen Validator auf Einhaltung von Standards überprüft und festgestellte Fehler behoben.
